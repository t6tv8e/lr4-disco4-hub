## Adding a New Key to a Land Rover Discovery 4 Using the GAP IID Tool

*A step-by-step walkthrough of programming a spare key to a 2012 Land Rover Discovery 4 using an aftermarket diagnostic tool, saving hundreds of euros compared to dealer pricing.*

---

### 🔑 The Problem & The Solution

- **Starting situation**: Only one key came with the car; a second key needed to be programmed.
- **Cheap key source**: A replacement key fob was purchased on eBay for approximately **€15–20**.
- **Dealer cost avoided**: A Land Rover dealer would charge **~€400** just to program a new key; clearing a simple fault code alone costs **€70**.

---

### 🔧 The Tool: GAP Diagnostic IID Tool

- **Cost**: ~**€550**, connects via **Bluetooth** to an iOS or Android app.
- **Capabilities**: Full vehicle diagnostics — read/clear fault codes, engine data, key programming, service resets, ECU flashing, suspension calibration, and more.
- **Comparison to budget alternatives**: A cheaper tool (e.g., iCarSoft at ~**€150**) only covers fault reading/clearing, battery registration, and service resets — far more limited.
- **Value argument**: At €550, the tool pays for itself quickly vs. repeated dealer visits (one key programming job alone = €400).
- **Connection point**: The IID dongle plugs into the **OBD port under the steering wheel**.

---

### 📋 Key Programming Process

- **Prerequisite**: The original working key must be present during the procedure.
- **Navigation path in app**: `Body Control (BCM)` → `Add a New Key`
- **Discovery during process**: The tool revealed **2 keys were previously programmed** to the car, but only **1 valid key detected** — suggesting the previous owner lost or damaged the second key.

---

### ⚠️ Troubleshooting: E5 Error

- **First attempt failed** with an `E5 - Operation Failed` error, caused by an **ECU compatibility issue** on certain vehicles.
- **Fix**: Email **gapteam@gap-diagnostic.com** with your ECU info (exported directly from the IID tool via `ECU Info` menu); the GAP support team responds quickly with a custom solution.
- **Resolution steps provided by GAP**:
  1. Update/flash the IID tool firmware via PC (one-button process).
  2. Re-connect to the car and flash the specific **KBM (Key Body Module) ECU** via `ECU Flash` in the app (~20 seconds).
  3. Re-run the key programming routine.

---

### ✅ Successful Programming Walkthrough

- After the ECU flash, the routine prompts: **"Press and hold unlock button on new key"** — holding the button on the new eBay key at the right moment completes the pairing.
- Process includes an **ECU initialization** (~15 seconds) and a **fault-clearing cycle**.
- Final verification: **start engine → turn off → restart** to confirm new key is fully active.
- **Result**: New key successfully unlocked and started the car.

### 🔮 Planned Future Uses for the IID Tool
